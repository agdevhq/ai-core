# core-ai Docs

This directory contains the Mintlify documentation for `core-ai`.

## Local Development

Install dependencies from the repository root:

```bash
npm install
```

Start the docs preview from the repository root:

```bash
npm run docs:dev
```

Check for broken links:

```bash
npm run docs:check-links
```

## Mintlify Monorepo Configuration

In the Mintlify dashboard, set the documentation path to `/docs` and enable the monorepo toggle.
